# UI Test Artifacts (GitHub Pages)

This branch is used **only** to host autogenerated UI test reports
(Playwright HTML) via **GitHub Pages**.

### What this branch is for
- Publishing Playwright test reports
- Storing historical test artifacts
- Serving reports as a static website
 
### What NOT to do here
- ❌ Do not add application source code
- ❌ Do not manually edit or delete files
- ❌ Do not open PRs against this branch

All content is managed automatically by CI workflows.

Please note - Artifacts have a 7 day retention policy(business days only). So will be auto-deleted after that.
